##################################
	Utilisation
##################################

Condition préalable :
	- Avoir installé Python 3 (ou version suprérieur)
	- L'ordinateur hôte doit être connecté à Internet (pour l'utilisation du mode multijoueur)
	- S'assurer que les 7 fichiers suivants son dans le même répertoire :
		main.py, CasesManager.py, ComputerPlaying.py, IconsManager.py, MultiPlayer.py, PossibleCombinations.py, Utils.py
	- S'assurer que le répertoire 'imgs' soit dans le même dossier que le main.py

Lancement du jeu :
	Pour lancer le jeu, il suffit d'exécuter le fichier 'main.py'

Contraintes techniques:
	Dans certains cas, le mode multijoueur ne pourra pas être utilisé, notamment sur les réseaux d'entreprises.
	Il est fort possible que le mode multijoueur ne fonctionne pas sur le réseau Wi-Fi de SUPINFO. 
	Il est donc conseillé d'utiliser son forfait 4G (en partage de connexion) ou un réseau domestique classique.

Erreur possible :
	Si le jeu refuse de se lancer, ou ne fonctionne pas correctement :
	- Placer les fichiers du jeu dans un répertoire situer a la racine de votre disque
	- Donner les droits de lecture et d'écriture au répertoire du jeu
	- Autoriser votre pare-feu à communiquer sur le port 1212
	- S'assurer qu'aucune autre application utilise le port 1212
	- Vérifier votre version de python

Compatibilité :
	Jeu testé sur: 
	- Windows 10 64b édition Famille
	- Windows 10 64b édition Pro
	- Linux débian 8 stretch avec Mint

Configuration minimale :
	Processeur Intel Pentium E2220 1 Mo ou AMD Ry 3
	1 Go de mémore vive
	50 Mo de stoquage disponible 
	Windows, Mac ou Linux (32 - 64 bits)
